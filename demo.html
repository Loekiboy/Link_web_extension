Link Web Extension - Demo

Deze demo toont het verschil tussen tekst voor en na het gebruik van de Link Web Extension.

Let op: Deze demo simuleert wat de extensie doet. In werkelijkheid werkt de extensie automatisch op alle websites zodra je deze installeert.

Hoe werkt het?

  1. De extensie scant automatisch alle tekst op de pagina
  2. URL's worden herkend met een geavanceerd patroon
  3. Gevonden URL's worden omgezet in klikbare links
  4. Links openen in een nieuw tabblad voor veiligheid
  5. Werkt ook met dynamisch toegevoegde content
