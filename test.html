Test Page for Link Web Extension

This page contains different shapes of plain text URLs so you can test the extension.

Test 1: Basic URLs with a protocol

Visit our website at https://www.example.com for more information.

Download files from http://files.example.com/downloads

FTP server: ftp://ftp.example.com/public

Test 2: URLs without a protocol (www)

Also visit www.github.com for open source projects.

More info at www.mozilla.org/en-US/ and www.chromium.org

Test 3: URLs inside a sentence

If you want to learn more, visit https://www.wikipedia.org or www.google.com for general information.

Test 4: Multiple URLs

Here are a few useful links: https://www.stackoverflow.com for programming questions, www.reddit.com for discussions, and http://news.ycombinator.com for tech news.

Test 5: URLs with parameters

Search at https://www.google.com/search?q=test&hl=en

Or visit https://example.com/page?id=123&user=test

Test 6: Existing links (should not be processed twice)

This is already a link: https://www.example.com

And this one too: Click here

Test 7: Code blocks (should be skipped)

    return fetch("https://api.example.com/v1/users");
